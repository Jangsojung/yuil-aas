### Backend 실행 방법 ---

1. cd backend
2. npm i
3. Backend 파일 .env 확인(DB, Port 확인필요)
   : 현재 PORT: 5001 / DB PORT 3307
4. npm run start

### Frontend 실행 방법 ---

1. cd frontend
2. npm i
3. npm run start

### Python 실행 방법 ---

1. cd python
2. pip install basyx-python-sdk==1.1.0
3. pip install flask== 3.1.0
4. pip install orjson==3.10.16
5. python main.py
